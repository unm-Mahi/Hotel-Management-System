# Hotel-Management-System
A Django web application built using the MVT architecture to manage employee records, image uploads, and database operations via SQLite.


**Project Description:**
Hotel Management System is a web-based application developed using Django that helps manage hotel operations such as room details, customer records, and bookings. The system provides an organized and efficient way to handle hotel data through a centralized backend interface.

**Technologies Used:**

1. Python
2. Django Framework
3. HTML, CSS, Bootstrap
4. SQLite3

**How to Run the Project:**

1. Install Python on your system.
2. Open the project folder in VS Code.
3. Create and activate a virtual environment.
4. Install required dependencies using pip install -r requirements.txt.
5. Run database migrations using python manage.py migrate.
6. Start the Django development server using python manage.py runserver.
7. Open http://127.0.0.1:8000/ in your browser.

( For deploying the Django project online, cloud platforms like Render, Railway, or PythonAnywhere can be used )
